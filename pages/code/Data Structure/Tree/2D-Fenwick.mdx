## 2D-Fenwick Tree

```cpp
template <typename T>
struct Fenwick2D { // 1-indexed
    vector<vector<T>> data;
    int FMAX;
    void init(int N, int M) { // Row: N / Column: M
        
    }
    void update(int idx, T v) {
        for(; idx <= FMAX; idx += idx & -idx) data[idx] += v;
    }
    T get(int idx) {
        T res = 0;
        for(; idx > 0; idx = idx & (idx - 1)) res += data[idx];
        return res;
    }
};

```

## Time Complexity

- Init: $O(N)$
- Update: $O(\log N)$
- Get: $(\log N)$

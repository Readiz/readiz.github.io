## Quick Sort

```cpp
template <typename T>
void qSort(int s, int e) {
    if (s >= e) return;
    int l = s, r = e;
    T pivot = data[(l+r)/2];

    while(l < r) {
        while(pivot < data[r]) --r;
        while(data[l] < pivot) ++l;
        if (l >= r) break;

        T tmp = data[l];
        data[l] = data[r];
        data[r] = tmp;

        ++l, --r;
    }
    
    qSort(s, r);
    qSort(l, e);
}
```


## Time Complexity

- Average: $O(N\log N)$
- Worst: $O(N^2)$

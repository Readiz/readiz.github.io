# 나만 보는 Mac 팁들

어언 나도 Mac 유저 2년차이다. 자꾸 기억을 까먹는 Mac 팁들을 정리한다.

## 스크린샷 단축키

- Cmd + Control + Shift + 3
  - 화면 전체를 캡쳐 + 클립보드로
- Cmd + Control + Shift + 4
  - 영역을 지정해서 캡쳐 + 클립보드로

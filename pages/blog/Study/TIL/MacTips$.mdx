# 나만 보는 Mac 팁들

어언 나도 Mac 유저 2년차이다. 자꾸 기억을 까먹는 Mac 팁들을 정리한다.

## 스크린샷 단축키

- Cmd + Control + Shift + 3
  - 화면 전체를 캡쳐 + 클립보드로
- Cmd + Control + Shift + 4
  - 영역을 지정해서 캡쳐 + 클립보드로

## PS 기초 셋업

추후 자세한 사항은 다시 좀 더 풀어서 쓸 필요 있지만 개괄적으로는 아래처럼 설정한다.

- CPH 설치
- Firefox 설치 (Codeforces Submit 용)
  - Browser Addon 설치
- GCC 설치
  - Brew 통해서 설치하면 간단함
  - 설치 이후 /opt/homebrew/bin/gcc 를 ln -s 명령어로 link 해서 사용도 가능하지만 시도는 안해봄
- Clang은 Debugging 시 사용가능하도록 설정할 것 (GDB 사용 불가, M1 기준)
- 이후 VS Code의 빌드 기본값을 해당 경로로 사용하도록 지정
- Template 같은 것은 따로 사용할 수 있게 설정 가능

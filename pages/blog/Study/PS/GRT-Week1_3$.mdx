## 문제 풀이

- 문제 제목: 합 계산기
- 문제 링크: https://level.goorm.io/exam/195685/%ED%95%A9-%EA%B3%84%EC%82%B0%EA%B8%B0/quiz/1

### 풀이 접근

이 문제의 경우는 조금 복잡해 보이지만, 결국 구현 문제이다. 그리고, `Python`에 내장된 `eval`을 활용하면, 생각보다 쉽게 구현이 가능하다.

명령어 수 N을 입력으로 받은 뒤, `eval` 함수를 통해, row 별로 계산 값을 구한다. 그리고 그 값이 소수가 되면 안되므로 `int` 형으로 변환한 뒤 합을 계속 계산하면 된다.

### 샘플 정답 코드

```python
N=int(input())
sum=0
for _ in range(N):
	cur=int(eval(input()))
	sum+=cur
print(sum)
```

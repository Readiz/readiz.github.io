## Simple isPrime

```cpp
bool isPrime(int num) {
    for(int i = 2; i * i <= num; ++i) {
        if (num % i == 0) return false;
    }
    return true;
}
```


## Time Complexity

- Simple isPrime: $O(\sqrt N)$

## DP 기본 이론 정리

- AtCoder Educational DP Contest 풀이 정리: <a href="/blog/History/AT/ATDP">링크</a>

구체적인 기본 유형 풀이는 위 링크에 정리 업데이트 예정이고, 여기는 해당 링크에 있는 유형의 키워드만 정리한다.

- Frog: 점프 거리 제한이 있는 DP
- Vacation: 같은 활동을 연속해서 못하는 DP
- Knapsack: 더 이상의 자세한 설명은 생략하는 DP
- LCS


### DP Signal

- 전체 경우의 수를 세라고 한다. 
- 답이 클 수 있으니 `10^9+7` 등으로 나머지를 구하라고 한다.
- 점화식을 세울 수 있다.
